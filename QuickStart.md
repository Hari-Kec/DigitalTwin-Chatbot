# Quickstart Guide

1. Clone the repo:
   git clone <repo-url> && cd Digital\ Clone

2. Install dependencies:
   pip install -r requirements.txt

3. Create env file:
   cp env_example.txt .env
   (Fill in API keys)

4. Run setup check:
   python test_setup.py

5. Start Streamlit app:
   streamlit run app.py
